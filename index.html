JavaScript

JavaScript is a programming language and core technology of the Web, alongside HTML and CSS. It was created by Brendan Eich in 1995. As of 2025, the overwhelming majority of websites uses JavaScript for client side webpage behavior. Web browsers have a dedicated JavaScript engine that executes the client code.

ReAct

React.js (also known as React or ReactJS) is an open-source front-end JavaScript library for building user interfaces (UIs), primarily single-page applications (SPAs). It is maintained by Meta (formerly Facebook) and a large community of developers.

Node JS

+8 Node.js is an open-source, cross-platform JavaScript runtime environment that executes JavaScript code outside of a web browser. Built on Google Chrome's fast V8 JavaScript engine, it enables developers to use a single programming language for both client-side (frontend) and server-side (backend) development, a concept known as "JavaScript everywhere".

MongoDB

MongoDB is a popular NoSQL document database that stores data in flexible, JSON-like documents (BSON) instead of traditional tables and rows, making it ideal for modern applications needing agility and scalability. It provides a dynamic schema, powerful querying, and horizontal scaling (sharding) for managing large datasets and high-throughput workloads, supporting diverse data types and evolving application needs without rigid structure
